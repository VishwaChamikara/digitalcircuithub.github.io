Running Light with Arduino

Control multiple LEDs to create a sequential running effect.

YouTube 
void setup() {
  for(int i=2; i<=7; i++) pinMode(i, OUTPUT);
}
void loop() {
  for(int i=2; i<=7; i++) {
    digitalWrite(i, HIGH); delay(100); digitalWrite(i, LOW);
  }
}
